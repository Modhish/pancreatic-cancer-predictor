## üöÄ Quick Start

### Prerequisites
- Node.js 18+ and npm
- Python 3.8+
- Git

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/your-org/pancreatic-cancer-predictor.git
cd pancreatic-cancer-predictor
```

2. **Setup Backend (Python/Flask)**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

3. **Setup Frontend (React/Vite)**
```bash
cd Frontend
npm install
```

4. **Environment Configuration**
```bash
# Create .env file in backend directory
echo "GROQ_API_KEY=your_groq_api_key_here" > backend/.env
echo "FLASK_DEBUG=False" >> backend/.env
echo "PORT=5000" >> backend/.env
```

### Running the Application

1. **Start Backend Server**
```bash
cd backend
python app.py
```
The API will be available at `http://localhost:5000`

2. **Start Frontend Development Server**
```bash
cd Frontend
npm run dev
```
The frontend will be available at `http://localhost:5173`

## üìä API Endpoints

### Core Endpoints
- `POST /api/predict` - Single patient prediction
- `POST /api/commentary` - Regenerate AI commentary for an existing result
- `POST /api/report` - Generate and download PDF report
- `GET /api/health` - Health check endpoint
- `GET /api/status` - System status and features
- `GET /api/model-info` - Model information and metrics

### Example API Usage

```javascript
// Single patient prediction
const response = await fetch('http://localhost:5000/api/predict', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    wbc: 5.8,
    rbc: 4.0,
    plt: 184,
    hgb: 127,
    hct: 40,
    mpv: 11,
    pdw: 16,
    mono: 0.42,
    baso_abs: 0.01,
    baso_pct: 0.2,
    glucose: 6.3,
    act: 26,
    bilirubin: 17
  })
});

const result = await response.json();
```

## üß¨ Medical Parameters

The system analyzes 13 key laboratory parameters:

| Parameter | Normal Range | Unit | Description |
|-----------|--------------|------|-------------|

| HGB | 120-160 | g/L | Hemoglobin |
